Curso de Diseño Web para Jóvenes

Clases en Vivo | Aprende HTML5 & CSS3 | Jueves 6:00 pm

Aprende Código Web de manera fácil y divertida
Comenzamos el Jueves 8 de Septiembre de 2016

La idea de este curso es invitar a jóvenes de 12 a 16 años a que desarrollen su primer página web utilizando los lenguajes HTML5 y CSS3 que son la base para la creación de todos los sitios web y muchas de las aplicaciones móviles que existen en el mundo.

¡Aprender Código es Genial!

Clases en Vivo

Toma una clase de 90 minutos en vivo a través de nuestra aula virtual una vez por semana.

Fácil, Práctico y Divertido

Aprende a través de lecciones dinámicas y crea una página web desde ceros con HTML5 y CSS3.

Actividades Semanales

Consulta la grabación de la clase y realiza actividades adicionales para aprender más.

Súbela a Internet

Aprende cómo subir tu propia página web a Internet y editarla cuando lo desees.

Aprender código web es el primer paso para adquirir superpoderes.

Temario del Curso

Semana 1: Herramientas

Semana 2: Principios de HTML5

Semana 3: Principios de CSS3

Semana 4: Principios de Diseño

Semana 5: Tu Primera Página Web I

Semana 6: Tu Primera Página Web II

Semana 7: Tu Primera Página Web III

Semana 8: Publicación de tu Página Web
